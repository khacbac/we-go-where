## Running the app

1. Clone the repository:

```
git clone https://github.com/khacbac/we-go-where.git
cd we-go-where
```

2. Install node_modules in project root directory:

```
yarn
```

### Running on iOS

Run directly from the command line:

```
yarn ios
```

### Running on Android

Run directly from the command line:

```
yarn android
```
